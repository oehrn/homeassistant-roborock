# Contributing

Thanks for contributing!

## Development
- Keep changes focused (one topic per PR).
- Prefer small, reviewable commits.
- Do not move or restructure `custom_components/roborock/` unless required.

## Pull Requests
Include:
- What changed and why
- Steps to reproduce / test
- Home Assistant version tested (if applicable)
- Logs/screenshots when relevant

## Release notes
If your change is user-visible, add a short note to `CHANGELOG.md` under "Unreleased".
